# Radio Silence — Simple Firewall to Block Internet for Individual Applications

## About Radio Silence

Radio Silence provides a lightweight application firewall that blocks internet access for selected applications — while all other apps remain connected. Radio Silence combines per-app network blocking, simple on/off toggle, background monitoring of connection attempts, and system integration (menu bar icon). Radio Silence enables users to stop apps from phoning home, sending analytics, checking for updates, or accessing the internet without permission. Radio Silence works with any application — background processes, helper tools, login items, and regular apps. Radio Silence is trusted by privacy advocates, offline workers, developers testing network failures, and anyone who wants to stop software telemetry.

## Deep Dive — Application Firewall

Per-app blocking — select any application from list or drag app icon to Radio Silence window  
Outbound connection monitoring — Radio Silence logs every blocked connection attempt (app name, destination, timestamp)  
Lightweight implementation — uses built-in network filter, minimal CPU/RAM usage, no kernel extensions  
Launch daemon integration — blocks background processes, helper tools, and login items that auto-start  
Whitelist/blacklist mode — block only selected apps, allow all others (default) or show all connections  
System integration — menu bar icon shows blocking status, quick toggle, recent block notifications

## Quick Reference for Radio Silence

| Operation | Radio Silence Action |
|-----------|----------------------|
| Add app to block list | Drag app icon from Finder to window → Click + |
| Remove app from block list | Select app → Click - |
| Toggle block on/off for app | Click checkbox next to app |
| Show blocked connection log | Click Log tab |
| Clear connection log | Log tab → Clear Log |
| Enable/disable Radio Silence | Menu bar icon → Enable Radio Silence |
| Quick block running app | Menu bar → Block Recent Apps |
| Show notifications for blocks | Settings → Show Notifications |
| Open preferences | Cmd+, |
| Quit Radio Silence | Menu bar → Quit |
